EINACHSER RACE CONTROL - GITHUB PAGES UPLOAD

1. Auf github.com ein neues oeffentliches Repository anlegen, z. B. einachser-race-control.
2. Im Repository: Add file -> Upload files.
3. Die Datei index.html aus diesem Ordner hochladen.
4. Commit changes bestaetigen.
5. Repository -> Settings -> Pages.
6. Unter Build and deployment: Source = Deploy from a branch.
7. Branch = main, Folder = / (root), Save.
8. Die danach im Bereich Pages angezeigte Adresse oeffnen.

Hinweis: Diese Testversion speichert Daten lokal im Browser. Fuer gemeinsamen Datenbestand aller Vereine ist spaeter eine zentrale Datenbank erforderlich.
